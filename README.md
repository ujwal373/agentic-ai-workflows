# Agentic AI Journey
Daily hands-on labs exploring planning, tools, memory, multi-agent, and eval.

## Structure
- `labs/` small focused exercises
- `projects/` weekly mini builds
- `packages/` reusable utils
- `eval/` smoke tests & datasets

## Environment
Python 3.11+
`pip install -r requirements.txt`

## Run
`python labs/01_tool_calling.py`
